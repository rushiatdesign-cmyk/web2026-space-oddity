# 07 — Katas 🥋 (homework)

**Three tiny drills.** A *kata* is a small movement you repeat until it's automatic — which is exactly what we're doing with flexbox. No visual polish expected; these are drills, not portfolio pieces. Each lives in its own file, and all three share `katas.css`.

Open `index.html` (in this folder) with Live Server to jump between them.

- **Kata 1 — `navbar.html`**: your name on the left, three links on the right, one row. The secret behind ten thousand websites: `justify-content: space-between`.
- **Kata 2 — `centered-card.html`**: a card centered horizontally *and* vertically in a full-height section. The legendary "center a div."
- **Kata 3 — `photo-row.html`**: three images sharing a row evenly, with gaps, that don't overflow on narrow windows. `display: flex` + `gap` + `max-width: 100%` on every image.

Write your CSS in `katas.css` — it's split into three commented sections, one per kata.

Stuck past your time-box? Stuck-note it. [Flexbox Froggy](https://flexboxfroggy.com) levels 1–13 cover everything these katas need, so revisiting the game is a legitimate solving strategy.
