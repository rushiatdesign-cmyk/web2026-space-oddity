# 01 — Ground Control 🛰️

**Your job: fix the CSS. Don't touch the HTML.**

Ground control has a problem. The mission console should be a neat row of five instruments — centered, evenly spaced. Right now it's a broken vertical pile.

Open `index.html` with Live Server. Then open `style.css`. There are four bugs, marked `BUG 1` through `BUG 4`. **Fix them one at a time and refresh after each**, so you can see exactly what each line does.

You'll touch four properties:

- `display` — is flexbox even on?
- `flex-direction` — row or column?
- `justify-content` — how do things spread along the row?
- `gap` — the space between children

When all four are fixed, the console snaps into a tidy control panel. That's the payoff.

**Concepts:** `display: flex`, `flex-direction`, `justify-content`, `gap`.
